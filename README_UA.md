# Теорія когнітивної еволюції

**Cognitive Evolution Theory** — робоча модель історичного розвитку людської когніції через чотири послідовні парадигми (Proto → Sapiens → Ludens → Digitalis), кожна з яких розгортається через чотири інваріантні вектори екстерналізації (Я–Ти–Ми–Вони).

> **Статус:** робоча гіпотеза / структурована модель для подальшої операціоналізації, **не** завершена й фальсифікована наукова теорія. Документ явно й послідовно позначає, де емпірична опора міцна, а де — спірна чи відсутня. Див. розділ «Обмеження» в [docs/theory.md](docs/theory.md#10-обмеження).

## Що це

Модель зіставляє 16 «модусів» когнітивної екстерналізації з конкретними, датованими історичними й науковими референтами — від палеоантропології (дзеркальний тест, Джебель-Ірхуд) до історії науки (Шеннон, 1948) — з явним розрізненням рівнів доказовості для кожного модусу.

## Структура матриці

| Парадигма | Я | Ти | Ми | Вони |
|---|---|---|---|---|
| **Proto** | Свідомість | Тіло | Рід | Природа |
| **Sapiens** | Розум | Знаряддя | Плем'я | Міф |
| **Ludens** | Символ | Текст | Ідеологія | Поліс |
| **Digitalis** | Знання | Технологія | Інформація | Мережа |

## Ключові тези

1. Після появи Homo sapiens головним кумулятивним механізмом розвитку стає не біологічна, а когнітивна еволюція.
2. Когнітивна еволюція відбувається через послідовну екстерналізацію функцій репрезентації, збереження й передачі знання.
3. Кожна історична когнітивна парадигма зберігає інваріантну чотиривекторну структуру (Я–Ти–Ми–Вони), змінюючи лише конкретний історичний зміст (модус) кожного вектора.
4. Нова парадигма не замінює попередню повністю, а розширює простір можливих когнітивних операцій — емпірично підтверджено співіснуванням, а не витісненням попередніх модусів (див. docs/theory.md, §10).
